# geekoops-mysql

Configurable ansible role for setting up a nginx webserver on a Linux server. Works with

- openSUSE Leap 15.4

## Role Variables
--------------

None at the moment.

## Example Playbook

Including an example of how to use the role:

---
- hosts: jellyfish
  user: root
  roles:
    - role: geekoops-mysql
